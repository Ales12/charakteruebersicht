Dies ist KEIN Plugin, sondern eine Reine PHP Datei. Das bedeutet, dass der Inhalt der Datei von euch per Hand angepasst werden muss.
Die SQL an sich muss nicht angepasst werden, diese sollte bei allen korrekt greifen.
Die Profilfelder etc. müssen von euch angepasst werden, da jeder andere Informationen braucht und die Profilfeld-IDs natürlich abweichend sind.

Der Plugin greift zudem auf Awesome Fonticons zu.
Stellte somit vorher sicher, dass diese auch in euren Forum funktionieren.
